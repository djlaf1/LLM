# Ultimate AI Chat - Transformer Architecture

A powerful command-line AI chatbot built with PyTorch transformer architecture, capable of understanding and discussing artificial intelligence concepts.

## Features

- **Transformer Architecture** with 768-dimensional embeddings
- **12 attention heads** and **12 transformer layers** for complex pattern recognition
- **1024 token context window** for extended conversations
- **Interactive CLI interface** with real-time training capabilities
- **Model persistence** - saves and loads trained models automatically
- **Comprehensive AI knowledge** - specializes in AI, machine learning, and neural networks

## Model Specifications

- **Architecture**:Transformer
- **Embedding Dimensions**: 768
- **Attention Heads**: 12
- **Transformer Layers**: 12
- **Context Length**: 1024 tokens
- **Vocabulary Size**: ~500 tokens (custom vocabulary)
- **Training**: Supervised learning with AI-focused dataset
